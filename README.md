# MsgNexus.AI

AI-powered messaging platform for unified inbox management.

## Features

- **Unified Inbox**: Manage all your messages in one place
- **AI-Powered**: Smart categorization and response suggestions
- **Gmail Integration**: Connect your Gmail account
- **Real-time Updates**: Stay on top of your messages

## Getting Started

See [SETUP.md](./SETUP.md) for detailed setup instructions.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Neon Postgres
- NextAuth.js
- OpenAI API
